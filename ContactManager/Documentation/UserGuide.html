Installation & Setup

System Requirements: Windows 10/11 (64-bit) - No additional software needed
1 Download Application: Extract ContactManager.exe from the deployment package
2 First Run: Double-click ContactManager.exe to launch the application
3 Database Creation: SQLite database (contacts.db) will be created automatically in the application folder
Important: Ensure the application has write permissions to its folder for database creation and Excel export functionality.

Main Application Window - Menu Bar, Data Grid, Control Buttons

Menu Options Description

  • Sortable Columns: Click any column header to sort ascending/descending

  • Auto-sizing: Columns automatically adjust to window size

Contact Management Operations

Adding New Contacts

1 Click the "Add" button or use Records ‚Üí Add menu
2 Fill in contact details in the popup form:
  • Name: Required field
  • Phone: Optional contact number
  • Email: Optional email address
  • Company: Optional company name
  • Used: Checkbox for usage status
3 Click "Save" to store the contact or "Cancel" to abort
Add/Edit Contact Dialog - Form Fields and Buttons

Editing Existing Contacts

1 Select a contact row in the data grid by clicking on it
2 Click the "Edit" button or use Records ‚Üí Edit menu
3 Modify the desired fields in the popup form
4 Click "Save" to apply changes

Deleting Contacts

1 Select a contact row in the data grid
2 Click the "Delete" button or use Records ‚Üí Delete menu
3 Confirm deletion in the confirmation dialog
Delete Warning: Deleted contacts cannot be recovered. The application will ask for confirmation before permanently removing any contact.

Excel Import & Export

Importing Contacts from Excel

1 Prepare your Excel file with contacts in .xlsx format
2 Use File ‚Üí Import Excel menu option
3 Select your Excel file in the file dialog
4 Choose whether to clear existing data or append to current contacts
5 Confirm import and view imported contacts in the data grid
Excel Format Expected:
Column A: Name
Column B: Phone
Column C: Email
Column D: Company
First row should contain headers

Exporting Contacts to Excel

1 Use File ‚Üí Export Excel menu option
2 Choose save location and filename in the file dialog
3 Click Save to create the Excel file with all current contacts
Export Features: Exported Excel files include professional formatting, headers, and all contact data including creation dates and usage status.

Advanced Features

Data Sorting

Click any column header to sort contacts:

  • First click: Sort ascending (A-Z, oldest first)
  • Second click: Sort descending (Z-A, newest first)
  • Available sort columns: Name, Phone, Email, Company, Used Status, Created Date

Visual Status Indicators

Row Color Status Meaning
Light Green Used = True Contact has been actively used or contacted
Light Red Used = False Contact is new or hasn't been used recently

Data Validation

  • Name Field: Required - cannot be empty
  • All Other Fields: Optional but validated for format when provided
  • Duplicate Prevention: System allows duplicate names but warns user
  • Data Integrity: SQLite constraints ensure database consistency

Error Handling

  • File Access Errors: Clear messages for permission or file lock issues
  • Database Errors: Automatic recovery and user notification
  • Excel Processing: Detailed error reporting for import/export issues
  • Validation Errors: User-friendly form validation messages

Troubleshooting

Common Issues

Problem: Application won't start
Solution: Ensure you have Windows 10/11 64-bit. Check that the application has execute permissions.
Problem: Excel import/export not working
Solution: Verify file permissions. Close Excel files in other applications. Ensure .xlsx format for imports.
Problem: Database errors
Solution: Check folder write permissions. Delete contacts.db to reset database if corrupted.
Problem: Application crashes
Solution: Restart application. Check Windows Event Log for detailed error information.
