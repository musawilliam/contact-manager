üìù Contact Management Operations

Adding New Contacts

1
Open Contact Form: Click the "Add" button or use Records ‚Üí Add menu option to open the contact creation dialog.
2
Complete Contact Details: Fill in the contact information in the popup form:
  • Name: Required field (cannot be empty) - First name or full name
  • Surname: Optional last name or family name
  • Number: Optional phone number or contact identifier
  • Used: Checkbox to mark contact as actively used (affects row color)
3
Save or Cancel: Click "Save" to store the contact in the database, or "Cancel" to abort without saving changes.
Contact Form Dialog - Clean Interface with Validation and Professional Styling

Editing Existing Contacts

1
Select Contact: Click anywhere on a contact row in the data grid to select it. The entire row will be highlighted to show selection.
2
Open Edit Dialog: Click the "Edit" button or use Records ‚Üí Edit menu. The contact form will open with existing data pre-filled.
3
Modify Information: Update any fields as needed. All existing data is preserved unless explicitly changed.
4
Apply Changes: Click "Save" to update the contact in the database. Changes are reflected immediately in the data grid.
Edit Features: When editing contacts, the form title changes to "Edit Contact" and all existing data is pre-populated. You can modify any field and the changes will be saved to the same database record.

Deleting Contacts

1
Select Target Contact: Click on the contact row you wish to delete. Ensure the correct contact is highlighted.
2
Initiate Deletion: Click the "Delete" button or use Records ‚Üí Delete menu option.
3
Confirm Action: A confirmation dialog will appear showing the contact name. Click "Yes" to permanently delete or "No" to cancel.
Deletion Warning: Deleted contacts are permanently removed from the database and cannot be recovered. The application displays the contact name in the confirmation dialog to prevent accidental deletions. Always verify you're deleting the correct contact.
Selection Validation: If no contact is selected when attempting to edit or delete, the application will display a helpful message prompting you to select a contact first.

üîß Troubleshooting & Support

Common Issues & Solutions

Problem: Application fails to start or shows .NET runtime errors
Solution:
  • Verify Windows 10/11 64-bit operating system
  • Install .NET 8.0 Runtime from Microsoft website
  • Check application has execute permissions
  • Run as administrator if using restricted user account
Problem: Excel import/export operations fail or show file errors
Solution:
  • Ensure Excel files are in .xlsx format (not .xls or other formats)
  • Close Excel files in other applications before importing
  • Verify read/write permissions to file locations
  • Check available disk space for export operations
  • Try saving Excel files to a different location
Problem: Database errors or data not saving
Solution:
  • Verify application folder has write permissions
  • Check disk space availability for database growth
  • Close other applications that might lock the database file
  • If corrupted, delete contacts.db to reset database
  • Run disk cleanup and check for file system errors
Problem: Application crashes or becomes unresponsive
Solution:
  • Restart the application and try the operation again
  • Check Windows Event Viewer for detailed error information
  • Ensure sufficient system memory is available
  • Close other resource-intensive applications
  • Update Windows and .NET runtime to latest versions
